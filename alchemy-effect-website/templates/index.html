{# Worker #}

Workers compose everything

A Worker is the entry point. It binds Durable Objects, Workflows, Dynamic Workers, and static assets — all as typed Effect dependencies. The type system ensures every resource is wired correctly.

  • Dependencies declared with yield*
  • Infra config and runtime in one class
  • Typed routing to DOs, Workflows, and more
export default class Api extends Cloudflare.Worker<Api>()(
  "Api",
  { main: import.meta.path, assets: "./assets" },
  Effect.gen(function* () {
    const rooms    = yield* Room;
    const notifier = yield* NotifyWorkflow;
    const agents   = yield* Agent;
    const loader   = yield* Cloudflare.DynamicWorker("Loader");

    return {
      fetch: Effect.gen(function* () {
        const req = yield* HttpServerRequest;
        // typed routing to DOs, Workflows, etc.
      }),
    };
  }),
) {}
{# Durable Object #}

Durable Objects hold state

Real-time WebSocket rooms, persistent sessions, and actor-model coordination. Each instance is isolated with its own state, and the API is fully typed — broadcast, fetch, and lifecycle hooks are all Effects.

  • WebSocket upgrade and session management
  • Hibernation-safe socket attachments
  • Cross-resource calls from Workflows
export default class Room extends
  Cloudflare.DurableObjectNamespace<Room>()(
  "Rooms",
  Effect.gen(function* () {
    return Effect.gen(function* () {
      const state = yield* Cloudflare.DurableObjectState;
      return {
        fetch: Effect.gen(function* () {
          const [res, ws] = yield* Cloudflare.upgrade();
          return res;
        }),
        broadcast: (text) =>
          Effect.gen(function* () {
            for (const p of sessions)
              yield* p.send(text);
          }),
      };
    });
  }),
) {}
{# Workflow #}

Workflows orchestrate across resources

Durable, multi-step workflows with named tasks, sleep, and cross-resource coordination. This workflow processes a message, broadcasts to a Durable Object room, waits, and finalizes — all with automatic retry and persistence.

  • Named durable tasks with typed payloads
  • Call into Durable Objects from Workflows
  • Durable sleep and automatic retries
export default class NotifyWorkflow extends
  Cloudflare.Workflow<NotifyWorkflow>()(
  "Notifier",
  Effect.gen(function* () {
    const rooms = yield* Room;

    return Effect.gen(function* () {
      const event = yield* Cloudflare.WorkflowEvent;
      const { roomId, message } = event.payload;

      yield* Cloudflare.task("process", ...);
      yield* Cloudflare.task("broadcast",
        rooms.getByName(roomId).broadcast(...));
      yield* Cloudflare.sleep("cooldown", "2 seconds");
      yield* Cloudflare.task("finalize", ...);
    });
  }),
) {}
